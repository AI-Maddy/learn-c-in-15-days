# Exercise 1 — Grade Checker

Write a C program that:
- Accepts a score between 0 and 100
- Prints:
  - "Pass" if the score is 50 or above
  - "Fail" otherwise

## Requirements
- Use relational operators (`>=`, `<`)
- Use logical operators if needed
- Validate that the score is within 0–100

## Example Output
Enter score: 72  
Pass

Enter score: 35  
Fail
