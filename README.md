# **Learn C in 15 Days — Hands‑On Roadmap**

A fast‑track, practical guide to learning the C programming language in just 15 days. This repository is designed for students preparing for exams, developers brushing up for interviews, and anyone who wants a crisp, example‑driven introduction to C.

The focus is simple: **short explanations, clear code snippets, cheat sheets, and daily hands‑on practice.**

---

## **Who This Is For**

- Students who want a structured revision plan  
- Beginners learning C for the first time  
- Interview candidates needing a quick refresher  
- Developers who prefer example‑first learning  
- Anyone who wants a compact, practical reference

---

## **How to Use This Repository**

You can follow it in two ways:

### **1. 15‑Day Learning Plan**
Move through one folder per day. Each day includes:
- A short concept summary  
- Example programs  
- Exercises  
- Common mistakes to avoid  

### **2. Quick Reference Mode**
Jump directly to:
- Cheatsheets  
- Code snippets  
- Interview questions  
- Topic‑wise folders

## **15‑Day Roadmap**

| Day | Topic | Folder |
|-----|-------|--------|
| 1 | Basics, setup, first program | `Day01_Basics` |

| 4 | Functions & modular programming | `Day04_Functions` |
| 5 | Arrays & strings | `Day05_Arrays_Strings` |
| 6 | Introduction to pointers | `Day06_Pointers_Intro` |
| 7 | Advanced pointers & pointer arithmetic | `Day07_Pointers_Advanced` |
| 8 | Structures & unions | `Day08_Structures_Unions` |
| 9 | Enums, typedef, header files | `Day09_Enums_Typedef` |
| 10 | File handling (I/O) | `Day10_Files_IO` |
| 11 | Dynamic memory allocation | `Day11_Dynamic_Memory` |
| 12 | Preprocessor & macros | `Day12_Preprocessor` |
| 13 | Data structures in C | `Day13_Data_Structures` |
| 14 | Advanced topics & best practices | `Day14_Advanced_Topics` |
