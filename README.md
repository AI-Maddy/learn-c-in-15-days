# **Learn C in 15 Days — Hands‑On Roadmap**

A fast‑track, practical guide to learning the C programming language in just 15 days. This repository is designed for students preparing for exams, developers brushing up for interviews, and anyone who wants a crisp, example‑driven introduction to C.

The focus is simple: **short explanations, clear code snippets, cheat sheets, and daily hands‑on practice.**

---

## **Who This Is For**

- Students who want a structured revision plan  
- Beginners learning C for the first time  
- Interview candidates needing a quick refresher  
- Developers who prefer example‑first learning  
- Anyone who wants a compact, practical reference

---

## **How to Use This Repository**

You can follow it in two ways:

### **1. 15‑Day Learning Plan**
Move through one folder per day. Each day includes:
- A short concept summary  
- Example programs  
- Exercises  
- Common mistakes to avoid  

### **2. Quick Reference Mode**
Jump directly to:
- Cheatsheets  
- Code snippets  
- Interview questions  
- Topic‑wise folders

## **15‑Day Roadmap**

| Day | Topic | Folder |
|-----|-------|--------|
| 1 | Basics, setup, first program, syntax, compilation | `Day01_Basics` |
| 2 | Operators & expressions, precedence, evaluation | `Day02_Operators_Expressions` |
| 3 | Control flow, branching, loops | `Day03_Control_Flow` |
| 4 | Functions, modularity, parameters | `Day04_Functions` |
| 5 | Arrays, storage, traversal | `Day05_Arrays` |
| 6 | Strings, manipulation, functions | `Day06_Strings` |
| 7 | Pointers, memory, arithmetic | `Day07_Pointers` |
| 8 | Dynamic memory, allocation, management | `Day08_Dynamic_Memory` |
| 9 | Structures, fields, organization | `Day09_Structures` |
| 10 | File handling, input, output | `Day10_File_Handling` |
| 11 | Unions, enums, typedef, representation, abstraction | `Day11_Bonus_Unions_Enums_Typedef` |
| 12 | Preprocessor, macros, directives | `Day11_Preprocessor` |
| 13 | Recursion, call stack, patterns | `Day12_Recursion` |
| 14 | Searching & sorting, algorithms, efficiency | `Day13_Searching_Sorting` |
| 15 | Linked lists, nodes, traversal | `Day14_Linked_Lists` |
| 16 | Advanced pointers, indirection, multi-level | `Day15_Advanced_Pointers` |
