# Memory Management Revision

## 1. Stack vs Heap
- Stack: automatic variables
- Heap: dynamic memory (malloc, calloc, realloc)

## 2. malloc
- Allocates uninitialized memory

## 3. calloc
- Allocates zero-initialized memory

## 4. realloc
- Resizes memory block

## 5. free
- Releases memory

## 6. Memory Errors
- Memory leaks
- Dangling pointers
- Double free
- Buffer overflow
